



## Initialiser nouveau dépot git avec un fichier déja existant
initialiser : git init 
ajouter :  git add .
import initial : git commit -m "Initial import"
ajouter origine : git remote add origin https://<votre_serveur_git>/<repository>.git
pousser : git push -u origin master (remplacer éventuellement master par main)
